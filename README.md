# Messing FE

Frontend application for Messing Chat, built with Vue 3 + Vite.

## Tech stack

- Vue 3
- Vue Router
- Pinia
- Axios
- STOMP over SockJS (real-time chat)

## Setup

```sh
npm install
```

## Development

```sh
npm run dev
```

## Production build

```sh
npm run build
```

## Preview production build

```sh
npm run preview
```
